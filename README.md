# Конфісклад - демо MVP мобільного додатку

Відкрити прототип:

```text
C:\Users\ALL\Documents\Конфісклад\index.html
```

Основний стартовий екран: `Каталог` з демо-асортиментом одягу та взуття.

Швидкі екрани для демонстрації:

```text
index.html
index.html?screen=store
index.html?screen=catalog
index.html?screen=orders
index.html?screen=chat
index.html?screen=profile
index.html?screen=cart
```

Нижнє меню: `Магазин`, `Каталог`, `Замовлення`, `Чат`, `Профіль`.

Це перший MVP для показу клієнту: fashion-каталог, кошик, статус замовлення, чат і бонусний профіль без прив'язки до 1С, платежів або обміну даними.
